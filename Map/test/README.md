# Description

Unit tests for ViewPort class.

1. Movements (pan left, right, up, down).

2. Centering on a target point.

3. Scaling.

4. Map2Screen (Transform map coordinates to screen coordinates)

5. GetLine

6. GetCircle

7. GetRectangle

# Tests

To run tests,

1. Install Google Test (`apt install googletest`)

2. `make`

3. `./tests`
